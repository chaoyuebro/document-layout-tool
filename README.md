# 排版工具 (Typesetting Tool)

A powerful web-based tool for formatting and styling text for various platforms like Zhihu, WeChat, and Feishu.

## Features

- **Multi-platform support**: Preview and format content for Zhihu, WeChat, and Feishu.
- **AI-powered**:
  - Smart typesetting
  - Image generation
  - Diagram creation
  - Content polishing
- **Real-time preview**: See how your content looks as you edit.
- **Customizable themes**: Choose from different themes to match your style.
- **Export options**: Easily export your content to the desired platform.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Code Editor**: @uiw/react-textarea-code-editor
- **AI Integration**: Custom AI service integration
- **Styling**: Tailwind CSS, Framer Motion

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/typesetting-tool.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
